📡 Basic Network Sniffer (Python)

A simple network packet sniffer built using Python and the powerful Scapy library. This tool captures live network packets and displays useful information like IP addresses, protocols, and payload data.

🚀 Features

📥 Captures live network packets

🌐 Displays Source & Destination IP

🔍 Identifies protocol type

📦 Shows packet payload (if available)

🧾 Prints packet summary for quick analysis

⚡ Lightweight and beginner-friendly

🛠️ Tech Stack

Python 3
Scapy (for packet sniffing)
📂 Project Structure
basic_network_sniffer_using_py.py
⚙️ Installation
1. Clone the repository
git clone
cd network-sniffer
2. Install dependencies
pip install scapy
